# bilans2json
Conversion des XML INPI des comptes/bilans des entreprises au format JSON


## Installation
Testée avec `go 1.16.2` et `go 1.17`
```
go get github.com/cquest/bilans2json
```

## Utilisation
```
~/go/bin/bilans2json $yourRef < yourFile.xml
```

`$yourRef` est pour le moment obligatoire permet d'introduire une clé `reference` dans le JSON de sortie pour aider le traçage des traitements.
